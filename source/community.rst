#########
Community
#########

***********************
About FreeUnit
***********************

FreeUnit is a community-maintained fork of NGINX Unit,
continued after the original project was archived in October 2025.
The project focuses on long-term security maintenance,
PHP 8.4+/8.5+ support, and community-driven development.

************
Discuss Unit
************

The `Discussions section <https://github.com/freeunitorg/freeunit/discussions>`_ of the
FreeUnit repo is the main place for discussing any topics related to the project.

Here you can share your ideas, ask questions, and discuss any issues both about
using FreeUnit and contributing to it.

*************************
Reporting bugs and issues
*************************

- You can report issues and bugs on the
  `Issues <https://github.com/freeunitorg/freeunit/issues>`_ section of the FreeUnit repo.

- If you have found a fix or want to suggest a change to the code,
  you can open a `pull request <https://github.com/freeunitorg/freeunit/pulls>`_
  on the FreeUnit repo.

***************************
Reporting security concerns
***************************

Please report any security-related issues concerning FreeUnit to
`team@freeunit.org <mailto:team@freeunit.org>`__.
For our mutual convenience, specifically mention **FreeUnit** in the subject and, if possible, provide a
`CVSS score <https://www.first.org/cvss/>`__.

************
Contributing
************

We welcome contributions to the FreeUnit project.

If you have a feature request or want to suggest an improvement, please submit a pull request on the
`FreeUnit repo <https://github.com/freeunitorg/freeunit/pulls>`_.

Please review the `Contributing Guidelines <https://github.com/freeunitorg/freeunit/blob/main/CONTRIBUTING.md>`_
before submitting your pull request.

You can also ask questions and discuss your suggestions by visiting the
`Discussions section <https://github.com/freeunitorg/freeunit/discussions>`_
of the FreeUnit repo.

*************
Documentation
*************

If you are interested in contributing to the **FreeUnit documentation**,
or find a typo or an error in the docs, you can submit a pull request on the
`docs repo <https://github.com/freeunitorg/docs/pulls>`_.
